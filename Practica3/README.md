## Actividades

- ¿Cuál es la diferencia entre Num e Int?

La diferencia esta en que `Int` (Integer) representa exclusivamente números enteros (sin decimales), mientras que `num` (numeric) es un tipo genérico que puede abarcar tanto enteros como números decimales, se podría decir que es un contenedor más amplio.

```sh
    num ---> conjunto de tipos numéricos
    int ---> un tipo especifíco de número entero
```

- Canción "El pollito pío" (Recursión)

Se podria decir que esta canción tiene cierta relación con la recursión ya que cada vez que se repite una estrofa, se agrega otro animal en la canción, comenzamos desde el pollo y se va agregando o contando nuevos como vendria a ser la gallina, el gallo, el pavo, etc... , entonces esta "estructura" va creciendo conforme se hace una llamada recursiva y asi hasta llegar a un caso base, en la canción podriamos decir que el pollito es el caso base y los demas animales algún caso recursivo.
