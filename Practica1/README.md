# Primera Práctica

## Objetivo de la práctica.

-En esta practica se tuvo como objetivo...Investigar sobre los lenguajes de programación y las diferencias que existen entre los demás lnguajes y el que vamos a usar mayormente en este curso el cual vendria siendo Haskell, ademas de conocer un poco mas las herramientas que utilizaremos no solo en la materia sino tambien a lo largo de nuestra vida profesional como computólogos.  

-Tiempo requerido en realizar la práctica completa: 1-3hrs.

-Comentarios y los problemas a los que te enfrentaste en la instalación: Buena práctica para introducirnos en el curso, no tuve ningún problema para realizar esta práctica.
